# CredLog### A minimal expense tracking app built with Kotlin & clean architecture.

A clean and lightweight Android app to track daily expenses with ease.  
Built using **Kotlin**, **MVVM**, and **Room**, featuring a smooth and modern **Material Design** UI.

---


## Features

- Add, edit, and delete expenses  
- Custom category picker with a styled dropdown  
- Local storage using Room Database  
- Modern Material UI with a minimal and clean look  
- Fast, responsive, and easy to use
- Easy to extend with new features (budgets, charts, etc.)

---


## Tech Stack

- **Kotlin**
- **MVVM Architecture**
- **Room Database**
- **ViewBinding**
- **Material Components**

---


##  Project Structure
```
app/
├── data/ # Room entities, DAO, repository
├── ui/ # Activities, adapters, viewmodels, fragments
├── utils/ # Helper classes
└── res/ # Layouts, drawables, themes
```
---


##  Getting Started

### Clone the project
```bash
git clone https://github.com/Code-Rrrrr/CredLog.git
```
Open the project in Android Studio, let Gradle sync, and run on any device/emulator.
